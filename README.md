# ai-configs

System prompts and configuration files to customize AI assistant behavior and context.

## Contents

- [financial-profile.md](financial-profile.md) - Financial preferences, portfolio information, and investment strategy guidelines
- [travel-profile.md](travel-profile.md) - Travel preferences, loyalty programs, and booking preferences

## Usage

These profile files can be used to provide context to AI assistants like Claude, helping them understand your preferences and provide more personalized assistance.
